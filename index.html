你好，我是文君豪，目前就讀於國立臺灣師範大學數學系。在修習代數學與線性代數的過程中，我培養了對純數學的熱愛與高敏感度。這份熱情驅使我利用暑假期間自修《Galois Theory》，並進一步探索同調代數的廣闊領域。


在深入純數之際，新加坡一直是我非常嚮往的城市。2018 年初次造訪時，我便對其先進的城市發展與高度自律的精神印象深刻；而 2025 年的再次拜訪，更成為我人生重要的轉捩點。當時我有幸與當地學生深入交流，他們展現出的極高效率、積極上進與對自我的嚴格要求深深打動了我，因為這些特質恰好與我對自身的期許不謀而合。


那段期間，我正好榮獲了 2025 校際應用數學秋季研討會「最佳論文獎」的殊榮。然而，在這份肯定的背後，未來的生涯走向卻帶給我極大的壓力。在審慎評估自身個性與嚮往的生活藍圖後，我對繼續深耕「純數」這條路產生了猶豫。這趟旅程宛如一劑強心針，回臺灣後，我更加確信新加坡就是我渴望置身的環境。新加坡擁有亞洲最頂尖的學府——新加坡國立大學與南洋理工大學，在各個前沿領域皆具備深厚的資源。如此完美的烏托邦怎會不讓人心動呢？


這份嚮往促使我立下了明確的目標：期望能進入這兩所頂尖學府之一，攻讀機器學習、資料科學與人工智慧相關的碩士學程。為此，我決心將專研領域從純代數向外拓展，期許能用這兩年的時間，淬鍊出一份足以說服頂尖學術機構的堅實背景。


恰逢人工智慧崛起的黃金年代，「如何讓無生命的機器學會思考？」這個問題深深吸引了我。在修習「科學計算導論」與「科學計算特論」後，我進一步掌握了以數據為核心、策略性解決問題的思維。憑藉著在代數領域的優勢，我開始從線性代數的視角切入，由研究 Graph Laplacian 起步，著手解決機器學習中的分類問題；隨後更實際融入圖卷積神經網路（GCN），設計出一套以資產配置為目標的投資演算法，正式跨足資料科學與機器學習領域。在參與多場工作坊與學術研討會並獲得肯定後，我更加堅定了朝此方向發展的信念。期許自己能持續將嚴謹的數學邏輯，轉化為解決真實世界問題的演算法，有朝一日在資料科學領域中，開創出屬於自己的一席之地。

代數學探討的是數學的「結構」本質。在撰寫程式或分析真實世界問題時，代數與線性代數能幫助我在腦中快速建構出清晰的數學模型。現實生活中線性關係隨處可見，尤其在電腦科學中，矩陣與向量空間的概念更是資料處理、圖論架構與現代機器學習最核心的底層邏輯。

機率與統計是量化「不確定性」的科學，為機器學習提供了嚴謹的數學視角。它使我能洞悉資料背後的分佈規律、精準評估風險；在面對真實世界中充滿雜訊或複雜結構的數據時，不僅能藉此建構出適應各種情境的預測模型，更能透過嚴密釐清各類誤差，為檢驗模型優劣提供客觀且扎實的判斷準則。

科學計算是連接純數理論與電腦實作的關鍵。在追求理論精確之餘，實務上更須妥善處理電腦運算時無可避免的誤差與極限逼近。它訓練我建立起以「運算成本」、「演算法效率」及「收斂性」為核心的思維模式，以嚴謹的數學知識，為解決真實世界問題的高效程式碼提供精準的想法與嚴謹的證明。

修習並研究中

泛函分析是探討函數空間與算子理論的核心數學分支。為了更透徹地理解 Spectral Theory 在 Graph Laplacian 中的底層邏輯，我展開了這門學科的自主研修。透過剖析無限維空間的性質與算子行為，我期望對機器學習中的最佳化問題具備更宏觀的視角，也為深入探究模型的泛化能力打下了堅實的理論底蘊。

作為實現資料科學的主力語言，自高中起便深耕的 Python 為我提供了強大且靈活的開發特性。在科學計算特論課程中，它在不同的技術層面皆提供強大的協助：從繁雜的資料清洗、建構複雜的模型，到最終的數據視覺化，皆一手包辦的它成為最強大的助攻之一。

學習 C 語言是對底層運算思維與記憶體管理的極致淬鍊，也是我在國立臺灣科技大學（NTUST）修習期間收穫最豐的技能之一。它訓練我以電腦硬體的視角剖析程式，深刻理解指令編譯與底層運行的原理，為日後實作高效能演算法與複雜資料結構，打下了無可取代的扎實基礎。

透過 IBM Data Science 專業認證課程的洗禮，我掌握了這項駕馭與萃取真實世界龐大數據的必備利器。對於具備嚴謹邏輯訓練的數學系學生而言，SQL 的關聯式思維極具共鳴，使我能以極高的效率，從海量資料庫中精煉出建模所需的關鍵特徵。

MATLAB 是我在大學期間深入掌握的重要數學語言。它提供了無與倫比的矩陣運算與數值分析環境；對我而言，它不僅是一台「超大型的科學計算機」，更是我在建構純數模型與快速驗證演算法原型時，最得心應手的強大引擎。

Data Science & Machine Learning

Machine Learning and Deep Learning +
Application of Graph Theory +

機器學習是我跨足資料科學的啟端。國立台灣大學有提供一門課程"機器學習導論" 只是我運氣不好沒有加簽到 不過我仍透過其課程錄影學到了大致的運作原理和核心架構。放眼未來，我將目標鎖定於新加坡國立大學（NUS）的資料科學與機器學習碩士學程 (DSML)，期許能在頂尖的學術殿堂中，建構更完整、深厚的機器學習知識體系。

圖論是我認為能將真實世界的複雜網絡與拓樸結構，最直觀轉化為數學語言的強大工具，而線性代數則是支撐其運作的核心基石。儘管未曾正式修習圖論課程，我卻在研究實務中將其概念深刻實踐。從深究 Graph Laplacian 的頻譜特性，到建構 Graph Convolutional Network (GCN) 與 Graph Attention Network (GAT) 以萃取複雜的節點特徵，圖論與機器學習的結合，是我目前最熱衷且持續突破的研究主軸。

研究與發表 / RESEARCH

Graph Laplacian & Spectral Clustering: Applications in Geometric Models, Urban Networks, and Medical Imaging

2026 | TWSIAM 台灣工業與應用數學會年會 海報發表

以 Stanford Bunny 幾何模型為例說明演算法核心，並將無從證明的社會科學理論（如都市經濟學）融入現代機器學習方法加以實證。後續更將此演算法框架應用於 AI 醫療領域，透過精準辨識腦腫瘤影像，輔助臨床醫師進行診斷，期能有效降低醫療人員的工作負擔。

第二次研究在演算法中加入特徵提取工程 (Feature Extraction Engineering)，將腦部MRI影像分為腫瘤區與非腫瘤區，後續研究欲引入Dice Coefficient並持續改良，期許未來開發輔助臨床醫師使用。

Factor GCN for End-to-End Portfolio Optimization

2026 | International Conference on Optimization and Machine Learning 海報發表

為克服傳統投資組合易受『假性相關』干擾，以及 HRP 模型受限於歷史價格雜訊的痛點，本研究提出基於圖論架構的 Factor GCN 模型。該演算法透過圖卷積神經網路，將長期基本面因子與短期動能特徵進行深度融合，有效過濾市場雜訊，並以端到端（End-to-End）的方式動態輸出最佳資產配置權重，最大化每單位風險之報酬。
